# نظام إدارة المبيعات - Streamlit

هذا التطبيق هو نسخة ويب من نظام إدارة المبيعات، يعمل على الهاتف والحاسوب عبر المتصفح.

## التثبيت المحلي

1. قم بتثبيت Python إذا لم يكن مثبتاً.
2. قم بتثبيت المكتبات المطلوبة:
   ```
   pip install -r requirements.txt
   ```

## التشغيل المحلي

```
streamlit run streamlit_app.py
```

سيفتح التطبيق في المتصفح على http://localhost:8501

## النشر على الإنترنت

### باستخدام Streamlit Cloud (مجاني وسهل):

1. **أنشئ حساباً على GitHub** إذا لم يكن لديك.
2. **أنشئ repository جديد** على GitHub وارفع جميع الملفات (streamlit_app.py, requirements.txt, runtime.txt, README.md).
3. **اذهب إلى [Streamlit Cloud](https://share.streamlit.io/)** وأنشئ حساباً (يمكن استخدام GitHub للتسجيل).
4. **انقر على "New app"** واختر repository الخاص بك.
5. **حدد الملف الرئيسي:** streamlit_app.py
6. **انقر على Deploy** وسيتم نشر التطبيق تلقائياً.
7. **احصل على رابط التطبيق** (مثل: https://your-app-name.streamlit.app)

### ملاحظات مهمة للنشر:

- **الأمان:** التطبيق محمي بتسجيل الدخول، لكن تأكد من تغيير كلمة المرور الافتراضية.
- **قاعدة البيانات:** في Streamlit Cloud، قاعدة البيانات SQLite محلية وستُعاد إنشاؤها عند كل تشغيل. للبيانات الدائمة، استخدم قاعدة بيانات سحابية مثل SQLite Cloud أو PostgreSQL.
- **HTTPS:** Streamlit Cloud يوفر HTTPS تلقائياً.
- **التحديثات:** أي تغيير في الكود على GitHub سيتم نشره تلقائياً.

### بدائل أخرى للنشر:

- **Heroku:** أكثر تعقيداً، يتطلب Procfile وإعدادات إضافية.
- **AWS/GCP/Azure:** للتطبيقات الكبيرة، يوفرون خدمات مجانية محدودة.

## تسجيل الدخول

- **اسم المستخدم:** admin
- **كلمة المرور:** password123

## الأمان

التطبيق يستخدم `streamlit-authenticator` للمصادقة الآمنة مع:
- تشفير كلمات المرور باستخدام hashing.
- ملفات تعريف ارتباط آمنة (cookies) للحفاظ على الجلسات.
- حماية من الاختراقات الأساسية مثل SQL injection (باستخدام parameterized queries).
- تخزين بيانات المستخدمين في قاعدة بيانات آمنة.

**للنشر العام:**
- استخدم HTTPS لتشفير البيانات أثناء النقل.
- قم بتشغيل التطبيق خلف خادم ويب آمن (مثل Nginx مع SSL).
- أضف CAPTCHA أو 2FA لمنع الهجمات الآلية.
- راقب السجلات للكشف عن محاولات الاختراق.
- حدث كلمات المرور بانتظام واستخدم كلمات مرور قوية.

## الميزات

- إدارة المنتجات: إضافة و عرض المنتجات.
- إدارة المبيعات: إضافة مبيعات مع كتابة اسم المنتج والسعر الإجمالي.
- إدارة الائتمان: تسجيل المبيعات الائتمانية للزبائن.
- إدارة المستخدمين: إضافة وعرض المستخدمين (للمدراء فقط).
- البحث في المبيعات حسب اسم المنتج.

التطبيق يستخدم قاعدة بيانات SQLite محلية.
